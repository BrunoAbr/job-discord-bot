# 🚀 Job Discord Bot

Bot desenvolvido em Node.js para buscar projetos no em sites de freelancers e enviar notificações para um webhook do Discord.

---

## 📌 Funcionalidades

- 🔎 Busca projetos por categoria
- 🧠 Renderiza páginas dinâmicas com Puppeteer
- 📤 Envia novos projetos para um canal do Discord
- 🔐 Usa variáveis de ambiente (.env) para proteger credenciais

---

## 🛠 Tecnologias Utilizadas

- Node.js
- Puppeteer
- Axios
- Dotenv
